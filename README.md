# FIFA 2026 World Cup

**An all-around statistics viewing tool for the FIFA 2026 World Cup.**

This is a small project that aims to do the following:

* Provide a clean user interface for engaging with the FIFA 2026 World Cup
* Provide real-time data for live games and past games
* **(WIP)** Provide predictions and player projections through an educated algorithm
